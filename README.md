# Web Push Toast Notification - CodingLab

Um componente de notificação flutuante do tipo Toast Popup (Web Push Notification) moderno, minimalista e responsivo desenvolvido com HTML5 e CSS3. O projeto simula um alerta discreto de inscrição posicionado no canto inferior direito da interface, ideal para engajamento de usuários e avisos de sistema.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do card de notificação, textos descritivos e botões de ação.
* CSS3: Estilização completa, uso de bordas arredondadas, sombras projetadas e posicionamento fixo.
* Flexbox: Organização e alinhamento horizontal dos elementos internos (logotipo, textos e botão).

## Funcionalidades e Técnicas Aplicadas

* Posicionamento Fixo Estratégico: Uso da propriedade `position: fixed` para fixar a notificação no canto inferior direito da tela, sobrepondo o conteúdo principal sem interromper a navegação.
* Logotipo Circular Estilizado: Inclusão de um avatar de marca redondo envolto por um gradiente multicolorido vibrante para atração visual imediata.
* Botão de Ação Direta: Botão "Subscribe" personalizado com cantos arredondados e cor azul ciano sólida de alto contraste.
* Controle de Fechamento: Um ícone discreto de fechar (X) posicionado no canto superior direito do card para permitir que o usuário descarte a mensagem.
* Efeito Flutuante Elegante: Uso de `box-shadow` suave sob o contêiner branco para criar profundidade em relação ao fundo claro.
